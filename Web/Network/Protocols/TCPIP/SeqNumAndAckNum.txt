SN: Sequence Number
	- [self_SN_1 = self_SN_0 	+ self_SN_0.SYN 	+ self_SN_0.FIN 	+ self_SN_0.DATA]
AN: Acknowlegement Number
	- [self_AN_1 = other_SN_0 	+ other_SN_0.SYN 	+ other_SN_0.FIN 	+ other_SN_0.DATA]

Initial SN values is 0 
	- The actual value is randomly set between (0 - 4294967295)
	- 0 is a relative value for convenience
	
Initial AN value is 0
	- This is actual value, since no package received yet
